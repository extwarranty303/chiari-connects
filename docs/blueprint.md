# **App Name**: React Refinery

## Core Features:

- Code Upload: Allow users to upload existing React (.jsx) code files to the application.
- Code Display: Display the uploaded code in an editable format within the application.
- AI Suggestion: Leverage a generative AI model to provide refactoring suggestions and improvements to the code.
- AI-Powered Tool Assistant: Integrate an AI-powered tool to analyze the codebase and provide contextual assistance based on project requirements and desired functionalities, reasoning when to apply the assistance.
- Component Generation: Generate new React components with AI assistance, including basic structure and functionality.
- Code Modification: Allow users to modify the code directly within the application.
- Code Download: Allow users to download the modified React code.

## Style Guidelines:

- Primary color: Indigo (#4B0082) for a sophisticated and modern feel.
- Background color: Light gray (#F0F0F0), slightly desaturated from the primary hue to provide a neutral backdrop that allows code to be read easily.
- Accent color: Violet (#8F00FF), a slightly brighter hue to call attention to certain sections without drawing excessive attention
- Body and headline font: 'Inter' (sans-serif) for clear, readable text in the code editor and other UI elements.
- Code font: 'Source Code Pro' (monospace) for consistent code rendering.
- Use simple, monochromatic icons for file operations, AI actions, and other controls.
- Subtle transitions when displaying AI suggestions and code modifications.
